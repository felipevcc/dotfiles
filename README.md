# My dotfiles

## Info 
* **OS:** Manjaro Linux 
* **WM:** Bspwm 
* **Bar:** Polybar 
* **Shell:** Zsh 
* **Compositor:** Picom
* **App Launcher:** Rofi 
* **Prompt:** oh-my-zsh (powerlevel10K)
